# 🍳 Recipes App

Aplicativo mobile desenvolvido em **Kotlin** para gerenciamento e visualização de receitas, com foco em organização, usabilidade e boas práticas de desenvolvimento Android.

---

## 📱 Sobre o projeto

O **Recipes App** é um aplicativo que permite ao usuário explorar receitas, organizadas por categorias, com informações como nível de dificuldade e detalhes de preparo.

O projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento mobile moderno utilizando Kotlin e arquitetura organizada.

---

## 🚀 Tecnologias utilizadas

* Kotlin
* Android SDK
* Jetpack Compose (UI moderna)
* Arquitetura em camadas (Model, Repository, UI)
* Navegação entre telas

---

## 📂 Estrutura do projeto

O projeto está organizado da seguinte forma:

```
app/
 └── src/main/java/br/com/fiap/recipes/
      ├── components/     # Componentes reutilizáveis de UI
      ├── model/          # Modelos de dados (Recipe, Category, etc.)
      ├── navigation/     # Rotas e navegação entre telas
      ├── repository/     # Camada de dados
      └── MainActivity.kt # Activity principal
```
